# Sample data

## `sample_mask.tif`

Binary segmentation of a vascular network, 704×640 px, uint8 (0/255).
Used by the Try-it section in the top-level README.

Sufficient detail for both the topological (`--remove-redundant all`,
default) and centerline (`--for-editor`) forms to be visually
distinguishable, and dense enough to exercise the editor at a
realistic node count.

Same license as the rest of the repo (MIT).
